## ğŸš€ Getting Started

### Prerequisites
- **Node.js** 18+ 
- **npm** 9+

### Installation

```bash
# Clone the repository
git clone https://github.com/Ronitdoes/Onerios-MUJ.git

# Navigate to the project
cd Onerios-MUJ

# Install dependencies

# Start the development server
```

